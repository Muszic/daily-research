# Metamorphic Testing with the Rashomon Set: Explanation Faithfulness in Machine Learning

- **Category:** Software Engineering
- **Date:** 2026-06-04
- **Link:** http://arxiv.org/abs/2606.06056v1

---
Here's a summary of the research paper using the Problem, Method, and Impact structure:

### Problem

Multiple machine learning (ML) models can achieve near-equivalent predictive performance on the same task, but provide *divergent feature-based explanations*. This phenomenon, known as the **Rashomon effect** in explainable AI (XAI), undermines trust and raises the critical question of which, if any, explanations are trustworthy. Current methods for assessing explanation faithfulness often require **ground-truth attributions** or synthetic data with known feature relevance, which are unavailable in real-world settings, creating an "oracle problem" for practitioners.

### Method

The authors propose a novel **metamorphic testing (MT) framework** to assess explanation faithfulness without requiring ground-truth labels. The method involves:

1.  **Constructing the ε-Rashomon Set:** Identifying a set of well-performing ML models whose test error is within a specified tolerance (ε) of the best observed error.
2.  **Defining Metamorphic Relations (MRs):** Five formal MRs are introduced to capture expected consistency properties between a model's predictive behavior and the feature attributions generated by a post-hoc explainer. These include:
    *   **MR1: Explanation Faithfulness:** If a feature is attributed higher importance, perturbing it should change the prediction more (single-model consistency). Metrics: Faithfulness Score (FS), Spearman rank correlation (FSrank), Area over the Perturbation Curve (AOPC).
    *   **MR2: Cross-Model Sensitivity Consistency:** If two Rashomon-set models agree on the most important feature, perturbing that feature should shift both predictions in the same direction (cross-model consistency). Metric: Rashomon Explanation Agreement (REA).
    *   **MR3: Explanation Divergence Implies Sensitivity Divergence:** If two models cite different top features, each should be more sensitive to its own cited feature than the other's (cross-model consistency for divergent explanations). Metric: Divergence Consistency (DC).
    *   **MR4: Invariance Under Irrelevant Transformations:** Perturbing features with near-zero attribution should not meaningfully change predictions or explanations. Metric: Explanation Fragility Index (EFI).
    *   **MR5: Proportional Attribution Response:** The ratio of output shifts from perturbing top features should approximately equal the ratio of their attributions. Metric: Proportionality Score (PS).
3.  **Testing Pipeline:** Applying the MRs by perturbing features of the Rashomon-set models, generating attributions from explainers (SHAP and LIME), and calculating aggregate metrics. Results are compared against a random-attribution null baseline.

The framework was demonstrated on two tabular regression datasets (California Housing, Wine Quality) using SHAP and LIME explainers across heterogeneous candidate pools of models.

### Impact

This work provides a **practical, model-agnostic, and oracle-free tool** for evaluating explanation faithfulness in machine learning. Key impacts include:

*   **Enables Trustworthy Model Selection:** Practitioners can select ML models not just based on predictive accuracy, but also on the reliability and trustworthiness of their explanations, even in real-world scenarios without ground-truth attributions.
*   **Reveals Explainer Behavior:** The framework empirically showed systematic differences in explanation faithfulness. Notably, LIME explanations were found to be nearly identical across heterogeneous Rashomon-set members (low DC applicability), suggesting it may homogenize explanations, whereas SHAP retained measurable cross-model divergence.
*   **Diagnoses Explanation Divergence:** The "Divergence Consistency (DC) applicability rate" is a novel diagnostic, indicating whether an explainer is suppressing genuine model diversity or if observed divergences are mere artifacts of the explainer itself rather than true differences in model strategy.
*   **Foundation for Future XAI Testing:** It lays the groundwork for further research into understanding the Rashomon effect's connection to explainable AI across more complex data modalities (e.g., images, text) and deep learning architectures.
